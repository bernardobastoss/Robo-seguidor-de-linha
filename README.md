## Descrição do Projeto:

O projeto consiste no desenvolvimento de um robô seguidor de linha, capaz de se locomover de forma autônoma seguindo um trajeto marcado no chão. Para que isso seja possível, o robô utiliza sensores infravermelhos para detectar a linha, de cor preta, e um sensor ultrassônico para identificar obstáculos, como paredes e objetos, tornando sua navegação mais segura e inteligente.
